# Custom Dart Code (FlutterFlow)

- **functions/**: Pure Dart functions (no side effects).
- **actions/**: Side-effect actions (DB writes, API calls).
- **widgets/**: Custom UI components bridging FlutterFlow gaps.

**Rules Documented:**
- **Source of Truth**: Maintain code here, copy-paste to FlutterFlow editor.
- **Naming**: Dart filenames must match FlutterFlow component names perfectly.
